#  Proyecto: KC - Cuadrado Mágico

# Características
*   Dinámico:Elige el tamaño del cuadrado y la tabla se adaptará al instante.
*   Cálculo Automático: Muestra la constante mágica esperada para cada orden.
*   Autocompletado: Incluye ejemplos automáticos basados en métodos clásicos (como el método siamés para órdenes impares).
*   Validación Inteligente:Detecta si faltan números, si hay duplicados o números fuera de rango,
   y pinta de color las celdas o líneas que tienen errores de suma.

#  Tecnologías utilizadas
*   HTML
*   CSS3 (Diseño adaptable, variables y animaciones nativas)
*   JavaScript (Vanilla) (Lógica de validación de matrices sin librerías externas)

# ¿Cómo usarlo?
1. Elige el orden del cuadrado en el menú desplegable.
2. Haz clic en "Trazar tabla".
3. Rellena los campos manualmente o haz clic en "Autocompletar ejemplo".
4. Presiona "Sellar y validar" para comprobar si es un cuadrado mágico.
